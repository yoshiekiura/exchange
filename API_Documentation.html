Cryptex API Documentation

Ether is token “0” at address “0x0000000000000000000000000000000000000000”

These can be called with no arguments.

  • https://api.cryptex.cc/returnTicker/
  • https://api.cryptex.cc/Trades/
  • https://api.cryptex.cc/Events/
  • https://api.cryptex.cc/orders/
  • https://api.cryptex.cc/topOrders/

The following can be called with arguments where [argument] is inserted without square braces []. Tokens are referenced by their address on the blockchain with Ether being token “0” at address “0x0000000000000000000000000000000000000000”

  • https://api.cryptex.cc/orders/[tokenA]/[tokenB]
  • https://api.cryptex.cc/events/[nonce]
  • https://api.cryptex.cc/events/[nonce]/[since]
  • https://api.cryptex.cc/topOrders/[nonce]
  • https://api.cryptex.cc/orders/[nonce]
  • https://api.cryptex.cc/orders/[nonce]/[tokenA]/[tokenB]

  • https://api.cryptex.cc/orders/0x0000000000000000000000000000000000000000/0xA8CDE321DDB903bfeA9b64E2c938c1BE5468bB75
